# yup-diff

```
NAME:
   diff - compare files line by line

USAGE:
   diff [OPTIONS] FILE1 FILE2

      Compare FILES line by line.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --unified value, -u value            output NUM (default 3) lines of unified context (default: 0)
   --context value, -c value, -C value  output NUM (default 3) lines of copied context (default: 0)
   --brief, -q                          report only when files differ (default: false)
   --ignore-case, -i                    ignore case differences in file contents (default: false)
   --ignore-all-space, -w               ignore all white space (default: false)
   --side-by-side, -y                   output in two columns (default: false)
   --recursive, -r                      recursively compare any subdirectories found (default: false)
   --help, -h                           show help
```
